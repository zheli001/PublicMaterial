# 第06章 开源与孤独

[目录](index.md) | [上一章](chapter_05.md) | [下一章](chapter_07.md)

---

开源像是一场公开的独白。你把代码放在平台上，期待有人回应，也许只收到冷静的 issue 或零星的 star。孤独并不可怕，可怕的是在无人回应时把自己的价值全部寄托于 PR 的反应数上。

我在开源社区学会了匿名的温柔：有人在凌晨帮你修了一个 bug，有人在 issue 下写了感谢，这些都是你虽不认识但能感受到的拥抱。开源还教会我透明：错误并不可耻，重要的是修复的过程。于是我开始在 README 里写更多关于失败的故事，把被踩过的雷都标注出来，像给后来者留一张地图。

孤独被社群的微光照亮。那微光不一定灿烂，但足够让你在深夜读到一句“我也遇到过这个问题”，然后心里不再那么空。
